To enable hardware accelerated rendering with OpenGL/JOGL, follow these steps:

All platforms:
  
  1. Download the current version of JOGL 1.x for your platform from
     
       http://download.java.net/media/jogl/builds/archive/jsr-231-1.1.1a/
       
  2. Extract lib/*.jar from the JOGL archive to
  
      $CINDERELLA_PLUGIN_DIR/jRealityPlugin
      
Windows:

  3. Extract lib/*.dll to the installation directory of Cinderella (on the same
     level as Cinderella.exe)
     
Linux:

  3. Extract lib/*.so to any directory in your LD_LIBRARY_PATH.
